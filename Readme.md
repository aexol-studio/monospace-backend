# imono.space GraphQL backend

## Installation

```
npm i
```

You will need environment variables to make it run
| Environment Variable | value |
|----------------------|-------|
| TWILLIO_SERVICE_ID | xxxxxxxxxx |
| TWILIO_ACCOUNT | xxxxx |
| TWILIO_TOKEN | xxxxx |
| JWT_SECRET | xxxxx |
| MONGO_URL | xxx |
| SPACES_KEY | xxxx |
| SPACES_SECRET | xxxx |

## Start

```
npm run start
```
